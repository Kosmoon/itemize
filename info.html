Getting started

Include script

CDN:

<script src="//cdn.jsdelivr.net/npm/itemizejs@1.11.4/build/itemize.min.js"></script>

Locally:

<script src="YOURPATH/itemize.min.js"></script>

Install with NPM

npm install itemizejs

Start coding!

var itemManager = new Itemize();
 itemManager.apply(".target_container");
